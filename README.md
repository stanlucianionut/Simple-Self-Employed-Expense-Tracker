**Overview**

This project is an Excel-based expense tracker designed to help self-employed individuals monitor business spending, categorize expenses, and maintain clear financial records. The tracker provides a structured layout for logging transactions and includes automated calculations to support budgeting and tax preparation.

**Features**

⦁	Expense Log: Organized table for entering dates, descriptions, vendors, categories, and amounts.

⦁	Automated Totals: Built-in formulas to calculate category totals, monthly totals, and overall spending.

⦁	Basic Validation: Consistent category selection to reduce entry errors.

⦁	Summary View: High-level overview of spending patterns to support budgeting and financial planning.

**How It Works**

1.	Enter each transaction in the expense log sheet.

2.	Assign each expense to a predefined category.

3.	The tracker automatically updates totals and summaries.

4.	Use these outputs for budgeting, invoicing, or preparing tax information.

**Use Cases**

⦁	Tracking daily business purchases

⦁	Monitoring monthly spending trends

⦁	Organizing deductible expenses for tax season

⦁	Maintaining simple self-employment financial records
